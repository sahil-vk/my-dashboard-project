# Cryptocurrency Data Analytics Dashboard

This project is developed and maintained by **Sahil Nechwani**.

It is a complete end-to-end cryptocurrency data analytics project that performs
both **real-time** and **historical** analysis of major cryptocurrencies and
presents the insights through an interactive web dashboard.

---

## What this project does

- Fetches **real-time cryptocurrency market data**
- Identifies the **top cryptocurrencies by market capitalization**
- Performs **historical analysis** on selected coins
- Generates insights such as:
  - Market cap distribution
  - Volume trends
  - Price movements
  - BTC dominance
  - Comparison of top coins vs rest of the market
- Displays everything in a **single interactive dashboard**

---

## How it works (high level)

- Python scripts automatically fetch and update data
- Data is stored as CSV files inside the repository
- GitHub Actions runs the update process on a schedule
- The dashboard reads the latest available data and visualizes it

Everything is automated — no manual updates required.
